Passos para usar o programa:
  1. usando cd até o diretório 'TP_Gabriel_130111350_Marina_110132351_Pedro_120132664' no terminal.
  2. digite make. Serão criados dois executáveis: teste e jogo e os arquivos funcoes.gcno e main.gcno 
    do gcov.
  3. com ./teste sera executado o teste do #CUnit
  4. com ./jogo sera executado o jogo em si. Também serão criados os arquivos funcoes.gcda e main.gcda.
  5. com #gcov funcoes.c será mostrado a porcentagem de linhas que são executadas do módulo funcoes.c.
    Além disso é criado funcoes.c.gcov que contém esta análise linha por linha.
  6. com #gcov main.c temos o mesmo para o módulo main.c. É criado main.c.gcov.
  7. com #vim funcoes.c.gcov temos a análise linha por linha.
  8. com vim main.c.gcov também.
  9. digite :q para sair.
  
  10. #Doxygen: com  doxygen -g será criado o Doxyfile.
  11. com doxygen Doxyfile são criados os arquivos de documentação HTML e Latex. Dentro da pasta html,
  abra o arquivo index.html para ver no browser.
  
  11. P/ usar o #splint. Escreva splint -weak +infloops +sysdirerrors +compdef nome_programa.c ou 
     nome_programa.h.
