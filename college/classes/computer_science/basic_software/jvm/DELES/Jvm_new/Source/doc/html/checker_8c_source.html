302  /*
303  Branches must be within the bounds of the code array for the method.
304  The targets of all control-flow instructions are each the start of an
305  instruction. In the case of a wide instruction, the wide opcode is
306  considered the start of the instruction, and the opcode giving the operation
307  modified by that wide instruction is not considered to start an instruction.
308  Branches into the middle of an instruction are disallowed.
309  No instruction can access or modify a local variable at an index greater
310  than or equal to the number of local variables that its method indicates it
311  allocates.
312  All references to the constant pool must be to an entry of the appropriate
313  type. For example: the instruction ldc can be used only for data of type int
314  or float or for instances of class String; the instruction getfield must

316  The code does not end in the middle of an instruction.
317  Execution cannot fall off the end of the code.
318  For each exception handler, the starting and ending point of code protected
319  by the handler must be at the beginning of an instruction or, in the case of
320  the ending point, immediately past the end of the code. The starting point
321  must be before the ending point. The exception handler code must start at a
322  valid instruction, and it may not start at an opcode being modified by the
