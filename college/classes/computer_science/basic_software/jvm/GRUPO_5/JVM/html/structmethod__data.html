Detailed Description

3.5.4 Method Area The Java virtual machine has a method area that is shared among all Java virtual machine threads. The method area is analogous to the storage area for compiled code of a conventional language or analogous to the "text" segment in a UNIX process.

It stores per-class structures such as the runtime constant pool, field and method data, and the code for methods and constructors, including the special methods (ยง3.9) used in class and instance initialization and interface type initialization.
